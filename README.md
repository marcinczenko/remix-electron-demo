# remix-electron-demo

Demo of [remix-electron](https://github.com/itsMapleLeaf/remix-electron) to use [Remix](https://remix.run) with [Electron](https://www.electronjs.org)
## Quick start

Installation: 

```bash
yarn install
yarn lint
yarn test
```

Developing:

```bash
yarn dev
```

Build desktop app executable:

```bash
yarn build
```
